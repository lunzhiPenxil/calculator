lg[]		|常用对数
sin[]		|正弦函数
cos[]		|余弦函数
tan[]		|正切函数
cot[]		|余切函数
sec[]		|正割函数
csc[]		|余割函数
arcsin[]	|反正弦函数
arccos[]	|反余弦函数
arctan[]	|反正切函数
